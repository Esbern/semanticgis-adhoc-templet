# Gemini Adapter

Use the canonical governance source in `ai/governance/`.

1. Read `ai/governance/root-policy.md` first.
2. Apply domain rules from `ai/governance/instructions/*.instructions.md`.
3. Use defaults from `ai/governance/workflow-preferences.yaml`.
4. Use website machine-readable contracts from `semanticgis.dk` / `semanticgis.org` for data-specific semantics.

If local override is requested, require explicit user confirmation and log reason, scope, and rollback in `Design_Rationale.md`.
